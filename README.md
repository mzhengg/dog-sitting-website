# Dog Sitting Website

A beautiful, modern website for a dog sitting business. Features:
- About Me section with images and descriptions
- Review section for customer ratings (1-5 stars) and comments
- Booking system with calendar for scheduling sittings
- Stripe payment integration for transactions
- Light pink and white color scheme

## Tech Stack
- React (frontend)
- Node.js/Express (backend)
- MongoDB (database)
- Stripe (payments)

## Setup Instructions
1. Install dependencies: `npm install` (in both frontend and backend folders)
2. Configure Stripe API keys and MongoDB connection string in environment files
3. Run backend: `npm start` in backend folder
4. Run frontend: `npm start` in frontend folder

## Customization
- Replace placeholder images and text in About Me section
- Update Stripe keys for live payments
